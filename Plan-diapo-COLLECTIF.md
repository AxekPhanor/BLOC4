# Plan de diapo — COLLECTIF (Projet expérientiel collaboratif)

> **Cadre** : groupe = membres de la DSI. Livrable = étude préliminaire / ébauche du schéma directeur du SI.
> **Format** : 20 min de présentation + 15 min de Q/R. Support type PPT, construit au fil de l'eau.
> **Notation** : grille collective `/20` (+ note individuelle orale `/12` par membre).
> Pondération entre `( )`. ⭐ = critères les plus pondérés. Argumentation = point central.

## Trame (≈ 15-16 diapos)

- [ ] **D1 — Titre** : « Manager les équipes et la transformation du SI », contexte Torpier, **noms des membres du groupe**
- [ ] **D2 — Contexte entreprise** : activité Torpier + **synthèse de la stratégie** d'entreprise (cadre les propositions)
- [ ] **D3 — SWOT de l'entreprise** : forces / faiblesses / opportunités / menaces clairement distinguées **(1)**
- [ ] **D4 — Cartographie des processus métiers** supportés par le SI + **tâches principales + acteurs** (représentation visuelle) **(2)**
- [ ] **D5 — Tableau applications du SI ↔ fonction(s) métier(s)** utilisatrices **(2)**
- [ ] **D6 — digramme de deploiemente** (serveurs + éléments actifs réseau) ↔ **applications supportées** **(2)**
- [ ] **D7 — Points forts / faibles du SI → 5 axes d'amélioration** cohérents avec le contexte **(1)**
- [ ] **D8 — Comparaison de 3 référentiels** (ITIL / COBIT / Scrum-Agile) : **avantages + limites pour Torpier** ⭐ **(2)**
- [ ] **D9 — Organisation globale des équipes SI** : relations **hiérarchiques + transversales** (le qui-fait-quoi) **(2)**
- [ ] **D10 — Matrice RACI** des responsabilités incluant les **parties prenantes clés** ⭐ **(2)**
- [ ] **D11 — Outils de suivi des projets** de transformation **(1)**
- [ ] **D12 — Outils de pilotage du changement** **(1)**
- [ ] **D13 — Pitch d'adhésion à la direction** : justifier l'adoption d'une politique de conduite du changement ⭐ **(2)**
- [ ] **D14 — Organisation du groupe de travail** : rôle + responsabilités de **chaque membre** sur l'étude **(½ de 1)**
- [ ] **D15 — Bilan du groupe** : points forts + axes d'amélioration **(½ de 1)**
- [ ] **D16 — Conclusion** + transition Q/R

## Forme (noté — 1 pt)
- [ ] Présentation **fluide et professionnelle**, support adapté à un **public IT / direction** **(1)**

## Garde-fous cohérence
- [ ] Même jeu de données que le plan individuel (mêmes 5 axes, même cartographie)
- [ ] **Argumenter** chaque proposition (conformité bonnes pratiques + référentiels)
- [ ] Le SWOT porte sur **l'entreprise** (pas sur le SI — le SI est traité via points forts/faibles à D7)

> ⚠️ **Hors périmètre collectif** (réservé à l'individuel) : schéma 4 strates détaillé, topologie réseau, 5 projets opérationnels, organigramme de l'équipe projet, plan de changement détaillé.
